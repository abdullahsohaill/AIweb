Browser Use Cloud enables both AI-powered and direct browser automation through six core concepts:
Plus standalone : Browser Sessions (direct CDP access) + Skills (reusable APIs)

## Core Concepts

Profile : Preserves login state and browser settings across automations Session : AI agent environment for natural language task execution Browser : Direct Chrome DevTools Protocol access for custom automation Task : Single automation instruction given to an AI agent Skill : Self-building API endpoint described in plain text Files : Input data for tasks and output results from agents

## Automation Approaches

### AI-Powered (Simple)

✅ Natural language ✅ Quick setup ❌ Limited control

### AI-Powered (Advanced)

✅ State preservation ✅ Multi-step workflows ❌ Limited control

### Direct Control

✅ Full control ✅ Custom scripts ❌ Requires programming

## When to Use What

AI-Powered Simple : Quick tasks, no login, proof of concepts AI-Powered Advanced : Multi-step workflows, login required Direct Control : Custom automation, integration testing Profiles : User-specific state, production apps
