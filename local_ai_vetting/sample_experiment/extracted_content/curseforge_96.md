## Configuration

- The direct config file is located at config/eval.json and can be hot reloaded via /eval reload
- Show the current eval config via /eval status
- The format pattern can be changed via /eval start '<pattern>' and /eval end '<pattern>'
- And inlining can be turned on and off via /eval inline on/off
