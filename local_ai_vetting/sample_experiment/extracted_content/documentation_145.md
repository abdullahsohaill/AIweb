## LLM.js

Run Large-Language Models (LLMs) 🚀 directly in your browser!
LLM.js provides JavaScript bindings for interacting with quantized large language models (GGUF).
Example projects🌐✨: Live Demo
[LINK: Live Demo](https://rahuldshetty.github.io/llm.js-examples/)
Models Supported:
- TinyLLaMA Series - 1,2,3🦙
- GPT-2
- Tiny Mistral Series
- Tiny StarCoder Py
- Qwen Models
- TinySolar
- Pythia
[LINK: Pythia](https://github.com/EleutherAI/pythia)
- Mamba and much more✨

## Features

- Run inference directly on browser (even on smartphones)
- Guidance: Structure responses with CFG Grammar and JSON schema
- Developed in pure JavaScript
- Web Worker to perform background tasks (model downloading/inference)
- Model Caching support
- Pre-built packages to directly plug-and-play into your web apps.
[LINK: packages](https://github.com/rahuldshetty/llm.js/releases)
