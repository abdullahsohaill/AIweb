## Treblle Integrations

All integrations automatically provide API monitoring, documentation generation, security analysis, and error tracking.

## API Gateway Integrations

Deploy Treblle at the infrastructure level for organization-wide visibility:

### AWS API Gateway

Serverless Lambda function for automatic API discovery across multiple AWS regions.
[LINK: View Documentation](/integrate-treblle/api-gateways/aws)

### Azure API Gateway

Event Hub and Azure Functions integration for enterprise Azure API Management environments.
[LINK: View Documentation](/integrate-treblle/api-gateways/azure)

### Kong

Plugin for Kong Gateway with request filtering and custom field masking.
[LINK: View Documentation](/integrate-treblle/api-gateways/kong)

### MuleSoft

Policy and auto-discovery app for enterprise MuleSoft environments.
[LINK: View Documentation](/integrate-treblle/api-gateways/mulesoft/policy)

### Traefik

Middleware plugin for Traefik proxy with route filtering capabilities.
[LINK: View Documentation](/integrate-treblle/api-gateways/traefik)

### WSO2

Global Synapse Handler integration for WSO2 API Manager.
[LINK: View Documentation](/integrate-treblle/api-gateways/wso2)

### Apigee

Shared flow integration for Google Cloud Apigee API Management platform.
[LINK: View Documentation](/integrate-treblle/api-gateways/apigee)
