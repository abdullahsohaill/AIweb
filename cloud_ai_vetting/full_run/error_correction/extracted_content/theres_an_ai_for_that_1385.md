MCP Integration

NextDocs now works as an MCP (Model Context Protocol) server, letting you create documents directly from your favorite AI assistants:

- Works with Claude (desktop, web, and API)
- Works with ChatGPT (via Apps SDK)
- OAuth 2.0 authentication for secure access

To try it, add this server URL in your AI assistant's settings:
https://mcp.nextdocs.io/mcp

---

Available Tools

NextDocs MCP provides three tools:

create_presentation
Creates canvas-based slide presentations. Perfect for pitch decks, sales presentations, training materials, and visual storytelling.

create_document
Creates professional documents — either canvas-based (fixed pages) or pageless (continuous flow). Great for resumes, proposals, reports, and whitepapers.

hello
A simple connectivity test that returns a greeting. Use it to verify your connection is working before creating documents.

---

Key Features

Multi-Variant Generation
Generate 2-4 design variations simultaneously. Each variant has different structure, visual direction, and layout approach. Compare side-by-side and pick your favorite.

AI-Generated Themes
Set theme_mode to "auto" (default) and NextDocs analyzes your prompt to generate custom color themes that match your content. A presentation about "Ocean Conservation" might get Deep Sea Blue, Aquatic Teal, or Marine Green themes.

Fast vs Quality Models
- fast (default): Optimized for quick generation, typically under a minute
- quality: More powerful models for complex content, better for detailed topics

Canvas vs Pageless Documents
- Canvas: Page-based with fixed dimensions, perfect for presentations and PDF exports
- Pageless: Continuous flow without page breaks, ideal for long-form content and articles

How to Connect

Claude:
1. Go to Settings → Connectors
2. Click "Add Custom Connector"
3. Enter server URL: https://mcp.nextdocs.io/mcp
4. Authorize with your NextDocs account
5. Start creating from any Claude conversation

ChatGPT:
1. Go to Settings and add a custom MCP connector
2. Enter server URL: https://mcp.nextdocs.io/mcp
3. Authorize with your NextDocs account
4. Start creating
