## FAQ

What is the Postman AI Agent Builder?
Postman's AI Agent Builder is a suite of tools that helps you build, test, and deploy agents and MCP servers powered by APIs from trusted publishers so you can create reliable, production-ready agents.
Do I need coding experience to use it?
No coding is required to get started. The builder includes templates and drag-and-drop workflows, but if you're technical, you can go deeper with custom logic, API calls, and model configurations.
What kinds of agents can I build?
If it involves APIs and automation, you can build it with an LLM in the loop. Check out our templates for agents in DevOps, Marketing, Sales, Customer Support and more.
Which APIs and LLMs are supported?
AI Agent Builder taps into Postman's network of over 18,000 APIs and supports models from top providers like OpenAI, Anthropic, and Meta.
Are there costs to use AI Agent Builder?
AI Agent Builder is free to use for building agents. Usage-based fees may apply if you deploy with Flows or integrate your own LLMs, depending on your volume.
