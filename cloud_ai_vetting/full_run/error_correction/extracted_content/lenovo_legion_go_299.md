## How to install SteamOS

### Description

This document outlines how to install the SteamOS OEM image on your platform.

### Applicable Brands

- Legion Go S 8APU1
- Legion Go S 8ARP1

### Solution

Step 1: Download the SteamOS image for your platform
Download the SteamOS IMG image file from the Lenovo support site. Visit Lenovo Support .
Step 2: Create a bootable USB key
Prepare a USB flash drive with a capacity of at least 16 GB. Note that this process will erase all data on the USB drive, so make sure you have backed up any important files.
The installation from Windows:
- Download and install Rufus from https://rufus.ie/ , using the portable version is acceptable.
- Unzip the file you downloaded and remember where you stored the .img file.
- Connect the USB drive to your PC.
- Launch Rufus and select your USB key within the Device field.
- Use the SELECT button to select your downloaded .img file.
- The default selections should all work. Click START .
- When prompted that your USB drive will be wiped, click OK to confirm.
- Allow the USB drive to be written; this process may take several minutes.
- Once completed, select CLOSE to finish the setup and safely eject the USB drive.
Step 3: Boot from USB flash driver and install SteamOS
- Insert the bootable USB drive into the installation platform while it is powered off.
- Hold the Vol+ button, then press the Power button to startup and access the Novo Button Menu .
- Tap Boot Menu on the touchscreen.
- Select the USB flash drive from the Boot Menu. The device will boot into the SteamOS repair environment. Note : The screen orientation may be incorrect, but this will not affect the recovery process. a. If a message indicates that the boot entry is blocked by the current security policy, navigate to BIOS Setup > Security , select Reset to Setup Mode under Secure Boot , press Enter, select OK in the dialog, and set Secure Boot to Disabled . If a message indicates that the boot entry is blocked by the current security policy, navigate to BIOS Setup > Security , select Reset to Setup Mode under Secure Boot , press Enter, select OK in the dialog, and set Secure Boot to Disabled . b. Go to the Exit tab, select Exit Saving Changes , and allow the device to reboot. If necessary, forcefully shut down the device by holding the power button, then attempt to boot from the USB flash drive again.

- On the desktop, click Reimage Steam Deck (not Reinstall SteamOS) and select Proceed in the pop-up window.
- After the installation completes, another popup will appear. Press Proceed , and the system will reboot. Remove the USB drive after the reboot.
Step 4: Complete the Initial Setup
- Select your language and time zone, then connect to a wireless network. The system will update to the latest Main channel version. Note : Make sure the network can connect to the Steam Network. If not, the update process will not proceed.
- After the first update completes, the device will reboot automatically. This may take approximately 10 minutes at the LEGION logo screen. Do not force a reboot or shutdown ; wait for it to proceed to the Steam login window.
- Log in to your Steam account and enter the Steam client.
