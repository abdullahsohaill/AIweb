## Stanford CS224W Tutorials 

The Stanford CS224W course has collected a set of graph machine learning tutorial blog posts , fully realized with PyG .
Students worked on projects spanning all kinds of tasks, model architectures and applications.
All tutorials also link to a Colab with the code in the tutorial for you to follow along with as you read it!
