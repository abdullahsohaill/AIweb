## WebRTC

WebRTC for Unity is a package that allows WebRTC to be used in Unity.
First, please check the requirements to make sure that the platform you are expecting is supported.
To get started, please see the Installation and Tutorial pages. The package also provides a number of samples. Check out the samples page to learn how to use them. And moreover, Unity Render Streaming also uses this package to provide a higher layer of API.
[LINK: Unity Render Streaming](https://docs.unity3d.com/Packages/com.unity.renderstreaming@latest)
