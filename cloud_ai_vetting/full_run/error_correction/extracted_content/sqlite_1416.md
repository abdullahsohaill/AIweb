SQLite is a C-language library that implements a small , fast , self-contained , high-reliability , full-featured ,
SQL database engine.
SQLite is the most used database engine in the world.
SQLite is built into all mobile phones and most computers and
comes bundled inside countless other applications that people
use every day. More Information...
SQLite source code is in the public-domain and is free to
everyone to use for any purpose.
