## Download the Magento Module:

### Downloading the AdScale Plugin...

If it haven't, click here.
Extract the plugin file, and using your FTP tool Upload the folder to the Magento Directory.
Run the following commands in PHP:
- 1. bin/magento module:enable AdScale_AdScale
- 2. bin/magento setup:upgrade
- 3. bin/magento setup:di:compile
- 4. bin/magento module:status
- 5. bin/magento setup:static-content:deploy
- 6. bin/magento cache:clean
Go to Magento Admin > Stores > Configuration and Click on AdScale > Enable
