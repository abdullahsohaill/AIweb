## Agentic Application Debugging

Neuron is the open source AI framework to create fully featured agentic applications in PHP - backed by Inspector. Neuron provides you with the most complete toolkit to build reliable agentic components into your existing PHP stack with easy. Working with AI powered apps can be even more challenging than working with normal code. LLMs can generate completely different outputs on the same inputs, and take decisions autonomously. Inspector allows you to look inside the workflow of your agentic systems in real-time.
