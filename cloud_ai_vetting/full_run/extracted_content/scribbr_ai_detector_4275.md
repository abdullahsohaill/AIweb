## Détecteur d'IA vs. logiciel anti-plagiat

Les détecteurs IA et les détecteurs de plagiat sont tous deux utilisés pour vérifier l’authenticité d’un texte. Leur mode de fonctionnement est toutefois différent.
Notre détecteur d’IA essaie de trouver des textes qui ont été créés à l’aide d’une IA comme ChatGPT, Copilot, Jasper ou Gemini. Pour cela, il mesure certaines caractéristiques du texte comme la structure et la longueur des phrases, le choix des mots et la prévisibilité. Le détecteur d’IA ne compare pas votre texte à une base de données.
Le logiciel anti-plagiat essaie de trouver des textes qui ont été copiés à partir d’une autre source. Pour cela, votre texte est comparé à une grande base de données de sites web, d’articles d’actualité, de magazines, etc. afin de trouver des similitudes. Le logiciel anti-plagiat n’analyse pas les caractéristiques du texte.
