### What is an AI translator and how does it work?

An AI translator uses neural machine translation to convert text or speech between languages, learning context from large datasets for more natural output. Modern tools support text, speech, images, and document translation. Typical engines cover 100 or more languages, with Google announcing 110 additional languages in 2024 and Microsoft listing broad language and speech options. Some tools translate full documents like PDFs, DOCX, and PPTX while keeping layout. To use AI well, compare two or three engines on a 150-300-word sample, then choose the output that handles names, numbers, and idioms best. For scanned PDFs, run OCR first. For meetings, evaluate latency, captions, and supported languages. Forum users recommend OCR and, for larger jobs, splitting files or using services like Google's Translation Hub. DeepL and other engines translate PDFs, DOCX, and PPTX.
