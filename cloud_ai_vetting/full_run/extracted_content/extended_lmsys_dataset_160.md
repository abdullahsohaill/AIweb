## Dataset Card for "LongChat-Lines"

This dataset is was used to evaluate the performance of model finetuned to operate on longer contexts. It is based on
a task template proposed by LMSys to evaluate attention to arbitrary points in the context. See the full details at https;//github.com/abacusai/Long-Context .
