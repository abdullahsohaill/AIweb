## Quickstart

First register an account which will generate for you an API key. Store that key somewhere safe - treat it like a password.
- If you do not want to register, you can use 0000000000 (10 zeros) as an API key to connect anonymously. However, anonymous accounts have the lowest priority when there are too many concurrent requests!
- To increase your priority you will need a unique API key and then to increase your Kudos. Read how Kudos work

### Ready to start creating?

The easiest way to use AI Horde is through one of our user-friendly frontends. We recommend starting with ArtBot - a popular web-based client that's perfect for beginners.

### Questions or need help?

Join our Discord server - it's the best and most active place to ask questions, get support, and connect with the AI Horde community!

## Frontends and Tools

### GUIs & Frontends

Easy-to-use applications for creating AI images and text. This includes web pages and mobile apps. Perfect if you just want to start making content.

### Workers

Software that lets you contribute your computer's power to help others. Earn kudos (priority points) by helping the community. Learn more in our FAQ.

### Developer Tools

SDKs, libraries, and plugins for developers who want to integrate AI Horde into their own projects.
Most users start with a GUI/Frontend. We recommend ArtBot for beginners - it's web-based, so there's nothing to install. Use the filters below to find the perfect tool for your needs. Have questions? Check our FAQ .

### Want to help power the Horde?

All the AI generation on this platform is provided by volunteers sharing their GPU power. Become a worker and earn kudos while helping make free AI accessible to everyone. Or, make a financial contribution to support the community.
