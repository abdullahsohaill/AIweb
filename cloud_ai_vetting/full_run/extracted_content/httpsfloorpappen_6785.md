## Unlimited Customization For Everyone

Enable more advanced customization than your default browser. Try Floorp, based on the smart Firefox, which enhances functionality and flexibility while maintaining security and privacy.

## What can Floorp do for you?

### Flexible UI Customization

Floorp's most distinctive feature is its customizability. Floorp is the first Firefox-based browser to enable UI customization, designed so that anyone can easily adjust their browsing experience to their preferences and access the web with ease.
[LINK: Learn More About Customization](https://docs.floorp.app/docs/features/design-customization)

### Various Ways to Browse the Web

Floorp features workspaces that integrate tab groups with containers that can separate login information. Additionally, you can easily access social media and chats through web panels.
[LINK: Learn More About Floorp's Unique Features](https://docs.floorp.app/docs/features/how-to-use-workspaces)

### Various Control Methods Supported

Browse more efficiently with mouse gestures and customizable keyboard shortcuts. Of course, it's also designed to be easily operated with a mouse.
[LINK: Learn More About Floorp's Gesture Features](https://docs.floorp.app/docs/features/how-to-use-gesture)

### Development & Research Focused

Floorp actively pursues experimental browser development. It's a browser that continuously incorporates new technologies and ideas, and energetically improves based on community feedback.
[LINK: Learn More About Floorp's Development System](https://docs.floorp.app/docs/values/research)

### Aiming for a Sustainable Web

We respect users' rights and contribute to building an open and sustainable web ecosystem. Let's create the future of the internet together for the next generation.
[LINK: What Floorp Is Doing](https://docs.floorp.app/docs/values/sustainable-web)

## About Us

### Floorp Project

Floorp is an open-source browser project derived from Firefox that started in Japan in 2021. Under the philosophy of 'user-centered', it achieves higher customizability and privacy protection, pursuing new possibilities in web browsing.
[LINK: GitHub](https://github.com/floorp-Projects)
[LINK: Documentation](https://docs.floorp.app)
We believe the web should be a place where everyone can freely access. Floorp aims to provide a better web experience while respecting users' privacy and cherishing the spirit of open source.
[LINK: Donate via GitHub Sponsors](https://github.com/sponsors/surapunoyousei)
You can participate in the Floorp project in various ways, including contributing to development, translation, and providing feedback. We value the spirit of open source and welcome contributions from the community.
