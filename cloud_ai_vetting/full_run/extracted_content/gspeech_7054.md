- What is Read Higlighted Text(RHT) Player, how to use it? The RHT Player is a very nice accessibility feature for any website to improve website accessibility, which allows all your website visitors to listen to any selected text on the webiste. It is being activated automatically after you create a website on dashboard! It also shows animated banners, which informs your website visitors about that(TTS) feature active. You can adjust one of 16 different templates , change banner position, texts, and much more. It has three possible views! Please check the following link, to see it in action: RHT Player Demo . To make the configuration, please go to: Dashboard->Website->Widgets->RHT Player. You will see the list of all options there.

- How to use Welcome Messages? Welcome Message is the audio greeting, which is being played once user visits your page. The widget Welcome Message is being created automatically when you create a website on GSpeech Dashboard, but it is unpublished by default ! To activate it, please go to: Dashboard->Website->Widgets->Welcome Message and set Status to Published . To change the welcome message: in the same page go to Content Tab and change the Content Text . To play only once: Go to Advanced Tab , and set Play Type to Once per session . To set up only on some pages, use Allowed Urls (under Content Tab ). Set one url per row. If you need only on homepage, place only url of homepage.
