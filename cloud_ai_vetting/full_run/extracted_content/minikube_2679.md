## Welcome!

minikube quickly sets up a local Kubernetes cluster on macOS, Linux, and Windows. We proudly focus on helping application developers and new Kubernetes users.
🎉 Latest Release: v1.37.0 - Sep 09, 2025 ( changelog )
[LINK: changelog](https://github.com/kubernetes/minikube/blob/master/CHANGELOG.md)

## Highlights

- Supports the latest Kubernetes release (+6 previous minor versions)
- Supports GPUs for AI development ( nvidia , amd , apple )
[LINK: nvidia](https://minikube.sigs.k8s.io/docs/tutorials/nvidia/)
[LINK: amd](https://minikube.sigs.k8s.io/docs/tutorials/amd/)
[LINK: apple](https://minikube.sigs.k8s.io/docs/tutorials/ai-playground/)
- Cross-platform (Linux, macOS, Windows)
- Deploy as a VM, a container, or on bare-metal or try in your browser .
- Multiple container runtimes (CRI-O, containerd, docker)
- Direct API endpoint for blazing fast image load and build
[LINK: image load and build](https://minikube.sigs.k8s.io/docs/handbook/pushing/)
- Advanced features such as LoadBalancer , filesystem mounts, FeatureGates, and network policy
[LINK: LoadBalancer](https://minikube.sigs.k8s.io/docs/handbook/accessing/#loadbalancer-access)
[LINK: network policy](https://minikube.sigs.k8s.io/docs/handbook/network_policy/)
- Addons for easily installed Kubernetes applications
[LINK: Addons](https://minikube.sigs.k8s.io/docs/handbook/deploying/#addons)
- Supports common CI environments
[LINK: CI environments](https://github.com/minikube-ci/examples)
