## The Materials Project

Harnessing the power of supercomputing and state-of-the-art methods, the Materials Project provides open web-based access to computed information on known and predicted materials as well as powerful analysis tools to inspire and design novel materials.

## The Materials Project by the numbers

## Search thousands of materials across dozens of properties.

Use the Materials Explorer to search through and view properties of all known solid-state materials , thousands more unknown materials , over 30,000 stable materials and over 6,000 materials with computed elastic tensors . Use our other explorers to find data on synthesis recipes , batteries , catalysts , and molecules

## View detailed computed information about any material.

Every material in our database has a detailed page displaying all of its
                                computed properties. By providing materials researchers with the information
                                they need to design better, the Materials Project aims to accelerate innovation
                                in materials research. See a random material.

## Access data directly using the Materials Project API

Anyone who registers with the Materials Project automatically receives free and
                                    unlimited access to our API. The API lets you pull data directly from our
                                    database and use it in your own applications. Python users can download the MPRester API client to make pulling data from the API even easier. Read more on our API page.
[LINK: Read more on our API page.](/api)
