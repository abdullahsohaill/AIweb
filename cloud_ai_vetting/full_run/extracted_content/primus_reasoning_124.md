## 🤗 Primus-Reasoning

The Primus-Reasoning dataset contains multiple cybersecurity reasoning tasks from CTI-Bench (CTI-RCM, CTI-VSP, CTI-ATE, CTI-MCQ). We use o1-preview to solve these tasks, generating reasoning steps and a reflection process . Additionally, we apply reject sampling, retaining only the samples with correct answers.
🚀🚀 For more details, see our paper: https://arxiv.org/abs/2502.11191
