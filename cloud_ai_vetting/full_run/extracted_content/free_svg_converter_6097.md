## Understanding SVG and STL Files

### What is an SVG?

A Scalable Vector Graphic ( SVG ) is an XML-based vector image format for 2D graphics. SVGs can be scaled infinitely without losing quality. Vector graphics software

### What is an STL?

An STL (Stereolithography) file is a standard format for 3D models, widely used in 3D printing and CAD. It describes surface geometry using triangles.

### Why Convert SVG to STL?

To give a 2D vector design depth, making it suitable for 3D printing or use in 3D modeling software.
