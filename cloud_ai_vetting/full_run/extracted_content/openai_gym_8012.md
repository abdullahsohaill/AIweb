## Gym Documentation

Gym has been unmaintained since 2022, and amongst other critical missing functionality does not support Numpy 2.0, and the documentation website has been taken offline.
Gymnasium is the maintained drop in replacement for Gym from the original Gym team if you're on the latest version of Gym.
Please upgrade your software to use Gymnasium, or contact the authors of your software and request that they upgrade.
You can read a migration guide for upgrading older version of Gym to Gymnasium here .
