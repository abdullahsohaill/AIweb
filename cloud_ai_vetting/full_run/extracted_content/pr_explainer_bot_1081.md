## 👋 Hello, I am PR Explainer

I'm an AI bot that helps you read and understand Pull Requests on Github faster.
Look what I can do:
- Explain changes in the pull request 📝
- Show suggestions on how to improve the code 💡
- Spot potential bugs 🐞
- Give you ideas for improvements 🤔
- Rate your code to see how you perform 💪

## How to use?

- Add the Github App to your personal account or organization
- Select a repository where you want to use me or add me to all repositories
- Create a pull request
- Wait for me to comment on your pull request
- Read my comments and improve your code
