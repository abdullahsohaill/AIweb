## How it works

### Text-to-image generation

Text-to-image object and style creation is the cornerstone of the Generative AI Product Design Starter Kit. It enables designers and customers to rapidly generate, edit, and render product designs based on textual descriptions.

### References, sketches, and masks

The starter kit provides advanced control and fine-tuning features that extend the basic text-to-image workflow. These features can be used to edit minor geometric and style details.

### Image-to-image workflows

The workflows can be created as sequences of image-to-image transformations with natural language guidance. This provides the flexibility to combine reference images, generated images, and textual prompts.
