The MiniWoB++ library contains a collection of over 100 web interaction environments ,
along with JavaScript and Python interfaces for programmatically interacting with them.
The Python interface follows the Gymnasium API
and uses Selenium WebDriver to perform actions on the web browser.
MiniWoB++ is an extension of the OpenAI MiniWoB benchmark ,
and was introduced in the paper Reinforcement Learning on Web Interfaces using Workflow-Guided
Exploration .
