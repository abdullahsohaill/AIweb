Creating Folders INSIDE Map Folders for layered organization.
D
Dree
Hello! Being able to group maps into different folders is very useful, but being able to create folders INSIDE those folders would be wonderful. Example of what I mean: Creating the initial campaign folder, then having folders inside that folder for maps of particular continents/regions then having folders inside that folder for maps of towns or cities in said regions and then a folder inside THAT folder for important buildings/landmarks or whatever. You get the idea. The problem right now is that I can create one folder and put maps in it, but then you cannot create a folder inside that folder for this sort of layered organization. 
Another example is I like to have a few variations of the same map, but the same issue occurs of having to use a new folder outside of the current folder they're in if I wanted to reduce the clutter. If I have multiple variations of the same map on multiple maps it begins to get cluttered and confusing very quickly if I keep it in the same folder.
If you're only making maps for one subject type/matter then the current system suits your purposes just fine, but for people like myself who are making maps for multiple different campaigns or stories etc. this feature would greatly help reduce clutter and would be a net benefit for map organization as a whole. 
Cheers!
