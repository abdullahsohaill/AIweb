## How It Works

## Fetching Transcribe from YouTube

Our model fetch the transcribe from the tedx video which is not yet punctuated

## AI Model Puncutate The Transcribed Data

The Transcribed data is then punctuated with the help of an AI Model.

## Converts Punctuated Transcribed Data into Summary

The Punctuated Transcribed data then sent to another AI Model Which Summarize it.

## About the Project

- Provides Nicely Punctuated TED Talk to Read.
- Provides short summaries for all the available TED Talk.
- Can select multiple Talks and compare their summaries.
