## Task The Modern Task Runner

A fast, cross-platform build tool inspired by Make, designed for modern workflows.
[LINK: Install](/docs/installation)
[LINK: Get Started](/docs/getting-started)
[LINK: Guide](/docs/guide)

## 30-Second Setup

Single binary download, zero dependencies. Works with Homebrew, Snapcraft, Scoop and more.

## Truly cross-platform

Run the same Taskfile on Linux, macOS and Windows. No extra setup. Task handles platform quirks so you don’t have to.

## Smart Caching

Skip unnecessary rebuilds by tracking file changes (timestamp or content-based).

## Ideal for code generation & scaffolding

Use Task to wire up codegen tools, formatters, linters, or anything repetitive. Chain commands, set dependencies, and keep your workflow clean.
