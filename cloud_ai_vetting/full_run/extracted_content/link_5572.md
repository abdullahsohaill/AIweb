Puppeteer is a JavaScript library which provides a high-level API to control
Chrome or Firefox over the DevTools Protocol or WebDriver BiDi .
Puppeteer runs in the headless (no visible UI) by default
[LINK: DevTools Protocol](https://chromedevtools.github.io/devtools-protocol/)
