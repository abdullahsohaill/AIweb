## Tips

- General Use (TTS and Voice Agents): The default settings ( exaggeration=0.5 , cfg=0.5 ) work well for most prompts. If the reference speaker has a fast speaking style, lowering cfg to around 0.3 can improve pacing.
General Use (TTS and Voice Agents):
- The default settings ( exaggeration=0.5 , cfg=0.5 ) work well for most prompts.
- If the reference speaker has a fast speaking style, lowering cfg to around 0.3 can improve pacing.
- Expressive or Dramatic Speech: Try lower cfg values (e.g. ~0.3 ) and increase exaggeration to around 0.7 or higher. Higher exaggeration tends to speed up speech; reducing cfg helps compensate with slower, more deliberate pacing.
Expressive or Dramatic Speech:
- Try lower cfg values (e.g. ~0.3 ) and increase exaggeration to around 0.7 or higher.
- Higher exaggeration tends to speed up speech; reducing cfg helps compensate with slower, more deliberate pacing.
Note: Ensure that the reference clip matches the specified language tag. Otherwise, language transfer outputs may inherit the accent of the reference clip’s language. To mitigate this, set the CFG weight to 0.
