There are a lot of fixes and improvements in this deployment. A more comprehensive list is featured on Discord, however the main change is the addition Providers in presets.
The Providers are re-usable connection details that you can easily create and switch between in your preset. At the moment the only place to create these are from within a preset. These will be added to the settings page soon. This is a fairly large change so let me know if you experience any issues with this feature or your existing presets. I've tried to test this quite thoroughly, but I always tend to miss something!
Over the coming days I intend to add some new Text+Image models and change some free tier models. This should include some vision models. Stay tuned!
[LINK: GitHub](https://github.com/agnaistic/agnai)
Some fairly large code changes are being deployment to try to improve and fix some issues mainly around message requests, especially for third party presets. Other notable changes with this release are:
- Added a new Claude called Claude (V2) service which supports images and reasoning options.
- Example Dialogue is now only inserted if the entire block fits.
- Preparing to add at least one vision model for subscribers
- Some fixes/changes to prefill and post behavior for OpenAI (Chat v2)
If you encounter any issues, please contact me so I can address it as quickly as possible.
Enjoy!
- Image Loras : Now available for Agnaistic image generation. If you'd like to request one, head over to the Agnai Discord.
- Stop Generation : This feature is finally available. Just hit the stop button next to the message input while a response is generating.
- Long-term Memory/Embeddings : These have had functionality and significant performance improvements. You can enable these in your site settings from the main menu.
- Character Editor : When creating a new character, your unsaved changes will be restored on refresh.
- Chat Images : These are now saved in your browser. You can also change the 'service and model' used to generate the image summaries in your Image settings .

## Update Next

I'll be looking into supporting Character expressions, adding Preset tags, 'Regex', and many other UI improvements this month.
Most of the post-performance improvement bugs have been found and fixed. Thank you everyone for your patience and bug reports. The improvement in performance was definitely worth it. While the bug fixing has taken place, there have been other changes and updates:
Chat Graphs When hovering or clicking a node a message preview is now displayed. The node labels have been replaced with the age of the message.
Image Generation More models have been added and the recommendations have been tweaked. A quick access button to the generation settings now sits near the Generate button in the character editor. I'll also be adding Prompt Prefix/Suffix and Sampler recommendations and a "use recommended settings" toggle to help get better results out of the gate. After that there will be image generation presets.
Presets Coming soon: The XTC and DRY samplers are being added.
Discord Mod Applications are Open If you are interested in being a vital part of the Agnai community, please consider applying! Find the application in the Announcements section of the Discord server.
A New Site is Coming! A new site that will run side-by-side with Agnai is coming soon. The site is for sharing characters, memory books, presets, etc. It'll be open to subscribers initially for beta and preview. Anyone can view, search, download, and rate. Subscribers will be able to publish. This is aimed to lessen the burden of moderation. Your 'private' character list will be tied to your Agnai character list. Published characters are kept separately. You'll be able to import characters straight to your Agnai account. The site will also feature it's own character editor very similar to Agnai.
Looking for help with getting started? Check out the Official Guide or head to the Agnaistic Discord .
