## FAQ

Nutanix Cloud Manager is a unified management platform that simplifies operations across hybrid multicloud environments. It provides capabilities such as intelligent monitoring, cost governance, self-service automation, and security compliance.
Nutanix Cloud Manager differs from other cloud management solutions in that it supports both hybrid and multicloud environments, including AWS, Azure, Nutanix, and VMware by Broadcom. It simplifies operations through seamless infrastructure integration, self-service, machine-learning-driven automation, and easy configuration.
Nutanix Cloud Manager offers multiple licensing tiers, including Starter, Pro, and Ultimate, each unlocking different levels of functionality. For detailed licensing and software information, visit the Nutanix Cloud Software Options page or contact us .
While Nutanix Cloud Manager (NCM) and Prism both offer user interfaces for managing Nutanix environments, Prism focuses on core infrastructure setup (such as cluster health and VM operations), whereas NCM builds upon those capabilities with advanced features such as automation, cost governance, and security compliance.
