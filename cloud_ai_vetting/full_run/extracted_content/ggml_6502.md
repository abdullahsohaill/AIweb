### GGML - AI at the edge

ggml is a tensor library for machine learning to enable large models and high performance on
                commodity hardware. It is used by llama.cpp and whisper.cpp
[LINK: ggml](https://github.com/ggml-org/ggml)
[LINK: llama.cpp](https://github.com/ggml-org/llama.cpp)
[LINK: whisper.cpp](https://github.com/ggml-org/whisper.cpp)
- Low-level cross-platform implementation
- Integer quantization support
- Broad hardware support
- No third-party dependencies
- Zero memory allocations during runtime
[LINK: ggml](https://github.com/ggml-org/ggml)
