## OpenAI.fm

OpenAI.fm is an interactive demo to showcase the new OpenAI text-to-speech models.
It is built with NextJS and the Speech API .
[LINK: Speech API](https://platform.openai.com/docs/api-reference/audio/createSpeech)
For more information about text-to-speech using the OpenAI API, check out our documentation .
[LINK: documentation](https://platform.openai.com/docs/guides/text-to-speech)
