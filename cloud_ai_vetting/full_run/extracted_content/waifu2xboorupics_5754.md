### After 8 years of running waifu2x without monetization we decided to use the GPU server for other projects.

### In all those years, AI image generation has advanced so rapidly that at this point there are better alternatives to waifu2x.

### Our site also runs slightly outdated models compared to the original. If you still want to use waifu2x please use the original generator https://www.waifu2x.net/

### Alternatively, you could use https://github.com/AUTOMATIC1111/stable-diffusion-webui to set up your own upscaler and AI image generator.

[LINK: https://github.com/AUTOMATIC1111/stable-diffusion-webui](https://github.com/AUTOMATIC1111/stable-diffusion-webui)
