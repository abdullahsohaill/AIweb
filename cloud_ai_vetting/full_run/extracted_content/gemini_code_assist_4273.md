**URL:** https://codeassist.google
**Page Title:** Gemini Code Assist | AI coding assistant
--------------------


## Gemini Code Assist: AI-first coding in your natural language

Uncap your potential and get all of your development done using Gemini Code Assist for individuals at no cost, or Gemini Code Assist for your business.

### Experience coding with fewer limits in your favorite IDEs

Gemini Code Assist brings the power of Gemini 2.5 to your IDE at no cost (with Gemini 3 coming soon), with a chat interface that is aware of your code, automatic code completions as you write, and generation and transformation of full functions or files on demand. A limit of 6,000 code-related requests and 240 chat requests daily can assist you through your tasks.

### Get Gemini in your terminal

Gemini Code Assist gives you access to Gemini CLI, an open source AI agent that brings the power of Gemini directly into your terminal. Gemini CLI provides powerful AI capabilities, from code understanding and file manipulation to command execution and dynamic troubleshooting, offering a fundamental update to your command line experience. While Gemini CLI provides a generous free tier with high usage limits for individual developers, you can supercharge your experience by using a Gemini Code Assist license.

### Accelerate development with AI agents

Gemini Code Assist pairs developers with AI agents capable of performing a wide range of actions across the software development life cycle, with support for multiple file edits, full project context, built-in tools and integration with ecosystem tools following Model Context Protocol (MCP), all while incorporating Human in the Loop (HiTL) for needed oversight.

### Reduce your code review time and improve code quality in GitHub

Gemini Code Assist for GitHub automatically reviews your pull requests in GitHub to find bugs and style issues, and automatically suggests code changes and fixes. By offloading these repetitive tasks, you can focus on what you love to do—write code. Anytime you need help, ask Gemini by adding a comment to your pull request with the /gemini comment.

### Get high quality code and chat responses with a large context window

Gemini Code Assist uses the code in your IDE, along with a context window of 1M tokens, to create responses that are more relevant to you. Customers using the Enterprise edition can connect their private source code repositories to get even more customized responses.

### Build a better app with AI-assistance in Firebase

By pairing Gemini Code Assist in the IDE with Gemini in Firebase, you get AI assistance across your entire development workflow, helping you build your mobile and web apps faster. Gemini in Firebase boosts productivity for individuals at no cost across Firebase surfaces with features like code generation, natural language chat, app error analysis, and performance insights.

### Prototype with a no-cost environment from Google Cloud

Google Cloud Shell Editor provides a no-cost, pre-configured development environment with Gemini Code Assist pre-installed. Cloud Shell Editor can be used 50 hours per week at no cost, allowing you to try out Gemini Code Assist for professional development and use it to build your next application.

Choosing which version of Gemini Code Assist to use for AI-powered coding assistance depends on your needs. See below for more details.

### Gemini Code Assist for individuals

If you're using Gemini Code Assist for yourself, consider starting here.
Developers can sign up with their personal Google account to access Gemini Code Assist available at no cost, no credit card needed. This version has high limits on operations such as code completions (6,000 per day), chat engagements (240 per day), and code reviews.

### Gemini Code Assist Standard and Enterprise

If you are using Gemini Code Assist for your team or business consider Gemini Code Assist Standard or Gemini Code Assist Enterprise. You can start with a 30-day trial , good for up to 50 people to use.
[LINK: 30-day trial](https://cloud.google.com/gemini/docs/discover/set-up-gemini#purchase-subscription)
These offerings feature enterprise-grade management tools, security, and generative AI indemnification. They require a Google Cloud account and a credit card or Cloud billing account.
