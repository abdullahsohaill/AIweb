# Aider Chat
**URL:** https://aider.chat
**Page Title:** Aider - AI Pair Programming in Your Terminal
--------------------


## AI pair programming in your terminal

Aider lets you pair program with LLMs to start
                        a new project or build on your existing
                        codebase.
[LINK: Documentation](/docs/)
[LINK: ⭐ GitHub Stars 39K](https://github.com/Aider-AI/aider)

## Features

[LINK: Cloud and local LLMs Aider works best with Claude 3.7 Sonnet, DeepSeek R1 & Chat V3, OpenAI o1, o3-mini & GPT-4o, but can connect to almost any LLM, including local models.](/docs/llms.html)

### Cloud and local LLMs

Aider works best with Claude 3.7 Sonnet, DeepSeek R1 & Chat V3, OpenAI o1, o3-mini & GPT-4o, but can connect to almost any LLM, including local models.
[LINK: Maps your codebase Aider makes a map of your entire codebase,
                        which helps it work well in larger projects.](/docs/repomap.html)

### Maps your codebase

Aider makes a map of your entire codebase,
                        which helps it work well in larger projects.
[LINK: 100+ code languages Aider works with most popular programming
                        languages: python, javascript, rust, ruby, go, cpp,
                        php, html, css, and dozens more.](/docs/languages.html)

### 100+ code languages

Aider works with most popular programming
                        languages: python, javascript, rust, ruby, go, cpp,
                        php, html, css, and dozens more.
[LINK: Git integration Aider automatically commits changes with
                        sensible commit messages.
                        Use familiar git tools to easily
                        diff, manage and undo AI changes.](/docs/git.html)

### Git integration

Aider automatically commits changes with
                        sensible commit messages.
                        Use familiar git tools to easily
                        diff, manage and undo AI changes.
[LINK: In your IDE Use aider from within your favorite IDE or editor.
                        Ask for changes by adding comments to
                        your code and aider will get to work.](/docs/usage/watch.html)

### In your IDE

Use aider from within your favorite IDE or editor.
                        Ask for changes by adding comments to
                        your code and aider will get to work.
[LINK: Images & web pages Add images and web pages to the chat to
                        provide visual context, screenshots, reference docs,
                        etc.](/docs/usage/images-urls.html)

### Images & web pages

Add images and web pages to the chat to
                        provide visual context, screenshots, reference docs,
                        etc.
[LINK: Voice-to-code Speak with aider about your code! Request new features, 
                        test cases or bug fixes using your voice and let aider
                        implement the changes.](/docs/usage/voice.html)

### Voice-to-code

Speak with aider about your code! Request new features, 
                        test cases or bug fixes using your voice and let aider
                        implement the changes.
[LINK: Linting & testing Automatically lint and test your code every time aider makes changes.
                        Aider can fix problems detected by your linters and test suites.](/docs/usage/lint-test.html)

### Linting & testing

Automatically lint and test your code every time aider makes changes.
                        Aider can fix problems detected by your linters and test suites.
[LINK: Copy/paste to web chat Aider works best with LLM APIs,
                        but it can also work an LLM via its web chat interface.
                        Aider streamlines copy/pasting code
                        back and forth with a browser.](/docs/usage/copypaste.html)

### Copy/paste to web chat

Aider works best with LLM APIs,
                        but it can also work an LLM via its web chat interface.
                        Aider streamlines copy/pasting code
                        back and forth with a browser.
