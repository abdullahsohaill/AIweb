- Distributed & Scalable Locust supports running load tests distributed over multiple machines, 
          and can therefore be used to simulate millions of simultaneous users

### Distributed & Scalable

- Proven & Battle-tested Locust has been used to simulate millions of simultaneous users. 
          Battlelog, the web app for the Battlefield games, is load tested 
          using Locust, so one can really say Locust is Battle tested ;).
