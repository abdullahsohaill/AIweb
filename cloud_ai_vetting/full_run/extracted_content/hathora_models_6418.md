## nvidia/parakeet-tdt-0.6b-v3

Multilingual automatic speech recognition model with word-level timestamps.

### Features

- Word-level timestamps
- Multilingual support
- High accuracy

## nvidia/parakeet_realtime_eou_120m-v1

Streaming speech recognition model with end-of-utterance (EOU) detection. Achieves low latency (80ms~160ms) and signals EOU by emitting an <EOU> token. Does not output punctuation or capitalization.

### Features

- Low latency EOU detection
- Streaming recognition
- English only

## hexgrad/Kokoro-82M

Kokoro is an open-weight TTS model with 82 million parameters.

### Features

- Lightweight architecture
- Fast inference
- Cost-effective

## ResembleAI/chatterbox

A public text-to-speech model optimized for natural and expressive voice synthesis.

### Features

- Natural voice synthesis
- Expressive speech
- Fast inference

## ResembleAI/chatterbox-turbo

A 350M parameter English text-to-speech model with paralinguistic tags (e.g., [laugh]) and lower compute / VRAM requirements—built for production voice agents.

### Features

- Paralinguistic tags ([laugh])
- Lower compute & VRAM
- English

## ResembleAI/chatterbox-multilingual

A 500M parameter text-to-speech model supporting 23+ languages with zero-shot voice cloning—ideal for global applications and localization.

### Features

- Zero-shot cloning
- 23+ languages
- Localization

## rime/arcana

A text-to-speech model for lifelike, emotionally rich voice synthesis.

### Features

- Expressive, natural-sounding speech with emotional nuance
- Fine-grained control over prosody, pacing, and tone
- Supports diverse accents and vocal demographics

## Qwen/Qwen3-30B-A3B

Qwen3 is the latest generation of large language models in Qwen series, offering a comprehensive suite of dense and mixture-of-experts (MoE) models. Built upon extensive training, Qwen3 delivers groundbreaking advancements in reasoning, instruction-following, agent capabilities, and multilingual support.

### Features

- Seamless mode switching
- Enhanced reasoning
- 100+ languages

## Qwen/Qwen3-Omni-30B-A3B-Instruct

Qwen3-Omni is a multimodal speech-to-speech model that can process images, audio, and text inputs and generate natural speech output. Perfect for building interactive voice assistants with visual understanding.

### Features

- Multimodal input (image, audio, text)
- Natural speech output
- Voice customization
