## Frequently asked questions

Codebase intelligence is a structured understanding of how repositories, modules, APIs, and dependencies relate to each other. Bito builds this intelligence so AI tools can reason about system level impact instead of working on isolated files.
AI Architect is Bito’s codebase intelligence layer exposed to AI coding tools through MCP. It lets agents query relationships across repos, services, and APIs, and it automatically supplies relevant system context during code generation and reviews using a continuously indexed view of the codebase.
Bito’s AI Code Review Agent uses AI Architect-powered codebase intelligence to analyze pull requests in context of the full system. Reviews include cross repo impact analysis, dependency awareness, and configurable rules that go beyond diff level feedback.
Bito indexes repositories as a connected graph, mapping symbols, modules, APIs, and dependency flows across repos. The index updates dynamically as code changes so system context stays current for coding and reviews.
Bito supports cloud and on-prem deployment for teams with different security requirements. Customer code is not stored and is not used to train models. Bito is SOC 2 Type II certified and designed for enterprise grade security and compliance.
