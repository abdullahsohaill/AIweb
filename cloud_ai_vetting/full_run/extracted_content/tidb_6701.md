## Rebuilt from the Core for Elasticity, Resilience, and AI-Driven Scale

Built on TiDB X, our brand new architecture, TiDB delivers cloud-native object storage, adaptive scaling, and modular design to power real-time, AI, and hybrid workloads with unmatched efficiency.

### Decoupled Compute & Cloud-Native Object Storage

Powered by the new TiDB X architecture, storage is now backed by cloud-native object stores for durability and elasticity, while compute scales independently to meet real-time demand.

### Context-Aware, Usage-Based Scaling

Autoscaling based on real-time workload signals and RU/s limits—no more overprovisioning or manual tuning.

### Modular Architecture for Resilience

Isolated system and customer services prevent cascading failures. TiDB stays available and responsive under stress.
