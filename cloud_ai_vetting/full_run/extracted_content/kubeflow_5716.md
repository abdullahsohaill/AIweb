### What is Kubeflow?

Kubeflow is the foundation of tools for AI Platforms on Kubernetes. AI platform teams can build on top of Kubeflow by using each project independently or deploying the
entire AI reference platform to meet their specific needs. The Kubeflow AI reference platform is
composable, modular, portable, and scalable, backed by an ecosystem of Kubernetes-native projects for each stage of the AI lifecycle . Deploy Kubeflow anywhere you run Kubernetes.

### Kubeflow Projects

Kubeflow Spark Operator aims to make specifying and running Spark applications as easy and idiomatic as running other workloads on Kubernetes.
[LINK: Kubeflow Spark Operator](/docs/components/spark-operator/overview/)
Kubeflow Notebooks lets you run web-based development environments on your Kubernetes cluster by running them inside Pods.
[LINK: Kubeflow Notebooks](/docs/components/notebooks/overview/)
Kubeflow Trainer is a Kubernetes-native project for LLMs fine-tuning and enabling scalable, distributed training across
a wide range of AI frameworks, including PyTorch, HuggingFace, DeepSpeed, MLX, JAX, XGBoost, and others.
[LINK: Kubeflow Trainer](/docs/components/trainer/overview/)
Kubeflow Katib is a Kubernetes-native project for automated machine learning (AutoML) with support for hyperparameter tuning, early stopping and neural architecture search.
[LINK: Kubeflow Katib](/docs/components/katib/overview/)
KServe is a standardized distributed generative and predictive AI inference platform for scalable, multi-framework deployment on Kubernetes.
[LINK: KServe](/docs/components/kserve/introduction/)
Kubeflow Model Registry is a cloud-native component that provides a single pane of glass for ML model developers to index and manage models, versions, and ML artifacts metadata.
It fills a gap between model experimentation and production activities.
[LINK: Kubeflow Model Registry](/docs/components/model-registry/overview/)
Kubeflow Pipelines (KFP) is a platform for building then deploying portable and scalable machine learning workflows using Kubernetes.
[LINK: Kubeflow Pipelines](/docs/components/pipelines/overview/)
Kubeflow Central Dashboard is our hub which connects the authenticated web interfaces of Kubeflow and other ecosystem components.
[LINK: Kubeflow Central Dashboard](/docs/components/central-dash/overview/)
