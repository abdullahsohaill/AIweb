## Open source alternative to AWS

Ubicloud provides cloud services on bare metal providers, such as Hetzner, Leaseweb, or AWS Bare Metal. You can set it up yourself or use our managed service

## Why use it?

Public cloud providers like AWS, Azure, and Google Cloud made life easier for start-ups and enterprises. But they are closed source, have you rent computers at a huge premium, and lock you in. Ubicloud offers an open alternative, reduces your costs, and returns control of your infrastructure back to you. All without sacrificing the cloud's convenience.

## Use cases

## Ubicloud PostgreSQL

Managed PostgreSQL that costs 3x less than comparable solutions. Open source and portable solution that helps you avoid cloud vendor lock-in.

## GitHub Actions Integration

Managed Ubicloud runners reduce costs by 10x. You change 1-line in your workflow file. You get isolated, clean, and ephemeral VMs for each job.

## Ubicloud Compute

Dedicated Virtual Machines (VMs) that cost 3x less. Encryption at rest and in-transit. Ideal for compute or memory heavy data processing needs.

## Services & Features

## Services & Features

Ubicloud provides IaaS cloud features on bare metal providers, such as Hetzner, Leaseweb, or AWS Bare Metal. You can set it up yourself or use our managed service.

## Elastic Compute

Provision, use, and delete isolated VMs on bare metal

## Virtual Networking

Public and private networking. IPv4 and IPv6. Flexible firewall rules. Network load balancer. Encryption in transit

## Block Storage

Block devices (non-replicated) with encryption at rest

## Attribute-Based Access Control (ABAC)

Define roles for different users. Provide fine-grained access control

## Managed PostgreSQL

Latest PostgreSQL versions. Automatic backups, point-in-time-restore (PITR), and high availability
