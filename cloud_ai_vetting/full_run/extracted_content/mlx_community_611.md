## MLX Community

A community org for MLX model weights that run on Apple Silicon. This organization hosts ready-to-use models compatible with:
[LINK: MLX](https://github.com/ml-explore/mlx)
- mlx-lm - A Python package for LLM text generation and fine-tuning with MLX.
[LINK: mlx-lm](https://github.com/ml-explore/mlx-lm)
- mlx-swift-examples – a Swift package to run MLX models.
[LINK: mlx-swift-examples](https://github.com/ml-explore/mlx-swift-examples)
- mlx-vlm – A Python package for inference and fine-tuning of Vision Language Models (VLMs) using MLX.
[LINK: mlx-vlm](https://github.com/Blaizzy/mlx-vlm)
- mlx-audio - A Python and Swift Package for text-to-speech (TTS), speech-to-text (STT) and speech-to-speech (STS) for MLX.
[LINK: mlx-audio](https://github.com/Blaizzy/mlx-audio)
These are pre-converted weights, ready to use in the example scripts or integrate in your apps.
