## Welcome to Upstage Console

Here, you can discover production-ready AI models developed by Upstage, to build your applications with ease.
If you're looking to leverage LLMs for complex document tasks—such as building a document QA service or extracting key information—you’ve come to the right place.
Concerned about privacy? Rest assured, we do not store or reuse any data sent to our APIs. For more details, check our terms .

## Start building

### Sign up to receive $10 in free credit

To sign up, press the Log in button at the upper right of this page.

### Get an API key

[LINK: Get an API key](#get-an-api-key)
To get an API key, just click the Create new key below. You can view keys in the API keys page .
[LINK: API keys page](/api-keys)

### Make an API request

[LINK: Make an API request](#make-an-api-request)
Choose a capability from the menu and make an API request using the provided example code.
Each capability includes a ready-to-use code snippet.
