Credits are the currency used on Markifact that allow you to run workflows. Each action costs a certain number of credits. All subscription tiers include a base number of credits. Node executions typically cost 0 credits by default, but nodes using paid APIs (especially AI nodes) cost more. Each workflow run costs 1 credit no matter how complex it is. You'll see credit costs displayed next to each node in the builder, with AI-powered nodes using the most credits. Additionally, Markifact Copilot (our AI agent that helps create and update workflows) costs 1 credit per message.
