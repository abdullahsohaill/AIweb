The Flesch algorithm is the most commonly used readeablity test.
The closer the index is to 100, the more readable the text.
Conversely, the closer the index is to 0, the less readable it is.
For most texts, a value between 60 and 70 is ideal.
The Gunning Fog algorithm is another famous method for evaluating the readability of a text.
The closer the index is to 0, the more readable the text.
Conversely, the closer the index is to 20, the less readable it is.
