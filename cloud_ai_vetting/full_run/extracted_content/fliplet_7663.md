- Smartphone, computer and tablet Apps work on all devices Fliplet apps are compatible with iOS, Android and web browsers. iOS and Android apps are compatible
                    with device features including camera, GPS and file storage so your apps work reliably online and
                    offline.
- Distribute your app everywhere Apple App Store, Google Play and web browsers Fliplet produces apps for iOS, Android and web. The apps can be launched to Apple App Store and
                    Android app stores including Google Play. They can also be used on modern web browsers.
- Capture data on the go Forms, camera, video, audio, QR codes Capture data in any format from your users with our extensive data capture features. Fliplet's form
                    builder enables apps to capture data quickly online or offline. Capture files, video, audio, text,
                    email, address, ratings, checkbox and radio, select box, typeahead and many more.
- Manage your data Powerful tables, import and export Fliplet's data tables look like spreadsheets and are easy to add, remove and modify your data. You
                    can copy and paste data from existing spreadsheets or upload data. Data can be secured and access
                    can be restricted to different types of users.
- View your data anywhere Store, view, search, filter, bookmark Display your data in apps using powerful lists with search, filter and bookmark capabilities.
                    Control what information is visible and how information is displayed in your apps. Lists
                    automatically show the latest data and work online or offline.
- Visualize your data instantly List, table and charts Build powerful dashboards with your data through lists, tables and charts. Lists can be customized
                    to show different data and offer search, filter and bookmark. Tables can be customised, filtered and
                    exported to CSV. Chart types include scatter, line, bar, column, pie and donut. Apps update
                    automatically as new data is available.
- Interactive, engaging features Forms, lists, maps, charts, and more Standard interactive features include forms, surveys, lists, maps, charts, sliders, and more.
                    Advanced interactive features include quiz, decision tree, assessment, comparison, task lists and
                    more.
- Notifications Push, email, SMS Apps can keep users informed through real time and scheduled notifications. Notifications can be
                    sent using push, email or SMS. Notifications can include content relevant to the user and link to
                    relevant content within the app helping users to respond to notifications quickly.
- Integrate with your existing systems Databases, files, APIs Database integrations including SQL Server. File integration including CSV import/export and file
                    upload. API integration with Sharepoint, Workday and more. Integration tools including Zapier, Logic
                    Apps and Power Automate.
- Security and roles Login, SSO, security rules, user types A suite of login features including username & password or SSO via SAML2 and oAuth. Apps include
                    security rules to control access to features and data. Configure different user types with different
                    access levels to featuers and data, such as Administrators, Managers and Standard.
- Control who can access your app Control access and distribution Apps can be publicly distributed via app stores and the web. Private apps can be distributed via
                    corporate app stores or as unlisted apps on public app stores. Access to apps can be restricted to
                    logged in users including integration with SSO so staff can use their existing corporate account.
                    Users can be assigned a user type to control their access to features and data within the app.
- Usage analytics and logs Monitor your apps usage App analytics includes feature, screen and user reporting. Weekly usage emails ensure you are kept
                    informed of your app's usage. App logs enable you to monitor what your apps are doing, detect issues
                    and take action.
