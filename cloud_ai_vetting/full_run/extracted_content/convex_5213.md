## The backend platform that keeps your app in sync

## Everything is code

From database schemas to queries, from auth to APIs, express every part of
        your backend in pure TypeScript. Your backend code lives next to your app code,
        is typechecked and autocompleted, and is generated by AI with exceptional accuracy.

## Always in sync

Convex libraries guarantee that your app always reflects changes to your
      frontend code, backend code,
        and database state in real time.
        No need for state managers, cache invalidation policies, or websockets.

## Backend built-ins

Create cron jobs, kick off backend AI workflows, leverage built-in auth,
        and tap into a growing ecosystem of components that solve common backend
        needs with just an npm i .
