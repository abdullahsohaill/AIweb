## Stability Matrix

Multi-Platform Package Manager and Inference UI for AI Image Generation

## One-Click Install

Get started instantly. No Python or prerequisite hassles.

## Bring Your Collection

Import models with drag & drop, plus metadata options.

## CivitAI Integration

Download the latest models seamlessly.

## Shared Models Folder

All models are shared between every web UI package, saving precious disk space. Import your existing models with drag and drop, with an option to include CivitAI metadata.
