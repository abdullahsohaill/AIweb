## Requests: HTTP for Humans™ ¶

Release v2.32.5. ( Installation )
Requests is an elegant and simple HTTP library for Python, built for human beings.
Behold, the power of Requests :
See similar code, sans Requests .
[LINK: similar code, sans Requests](https://gist.github.com/973705)
Requests allows you to send HTTP/1.1 requests extremely easily.
There’s no need to manually add query strings to your
URLs, or to form-encode your POST data. Keep-alive and HTTP connection pooling
are 100% automatic, thanks to urllib3 .
[LINK: urllib3](https://github.com/urllib3/urllib3)
