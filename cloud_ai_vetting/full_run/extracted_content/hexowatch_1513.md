## Be the first to know when important changes happen

With 13 monitoring types, Hexowatch makes it easy to stay informed when any page changes are detected - at scale.

### Visual monitoring

Monitor web pages for any visual changes. Select a part of the screen or the whole page.

### HTML element monitoring

Monitor specific HTML elements or the content of any field for changes.

### Keyword monitoring

Monitor web pages for the presence or absence of specific keywords.

### Technology monitoring

Monitor changes to the tech stack or 3rd party scripts used on the web page.

### Source code monitoring

Monitor web pages for any source code/HTML modifications.

### Content monitoring

Monitor the visible content on a web page for any changes.

### Availability monitoring

Monitor any URL for downtime, and get alerts when a web page goes down.

### Domain WHOIS monitoring

Monitor the ownership and administrative records of the domain names.

### Sitemap monitoring

Monitor any website for new or deleted pages via their sitemap.

### API monitoring

Monitor HTTP post and API endpoints for response changes.

### Backlink monitoring

Monitor your backlink partners and get alerts when links back to your website are removed or modified.

### RSS feed monitoring

Monitor RSS feed updates for changes.
