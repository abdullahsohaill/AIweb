### What is Airchat?

Airchat is a push-to-talk, voice-first public messenger with near-perfect transcripts.

### Is this the official Airchat web client?

No, this is an unofficial web client for the Airchat app, created as an open-source project which can be contributed to or used as an example for how to use the airchat api. The official Airchat app does not have a web client nor a public API. You can contribute to this project at https://github.com/aaln/airchat-web .
[LINK: https://github.com/aaln/airchat-web](https://github.com/aaln/airchat-web)
