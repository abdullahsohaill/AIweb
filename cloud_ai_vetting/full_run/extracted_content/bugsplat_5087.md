## All the tools you need, none of the complexity

### Rich Crash Context

Capture everything you need to fix a bug—logs, screenshots, even video clips around the crash. Add custom attributes like build, GPU, map name, or player state. It's all searchable and filterable—so you spot patterns faster.

### Symbolic Call Stacks

Readable call stacks with function names, line numbers, and file paths—plus local variable values and function parameters on Windows and Xbox. No hex soup. No guesswork.

### Intelligent Crash Grouping

BugSplat automatically groups crashes by the actual root cause—not just the top of the stack. That means fewer duplicate issues and a clearer view of what's really breaking.

### Customizable Crash Grouping

Our automatic grouping is smart, but every codebase is different. Define your own rules using function names, file names, and line numbers to make grouping match the way your team works.

### Lightning-Fast Search

Filter by version, user ID, platform, custom attributes—anything you want. Get to the crash you're looking for in seconds, no matter how big your dataset.

### Regression Detection

BugSplat automatically flags regressions when previously fixed crashes resurface in newer builds. No configuration required—it just works, so you can resolve regressions immediately.

### Crash Alerts

Get notified when a new crash type shows up, a regression is detected, or crash volume suddenly spikes. Alerts go straight to Discord, Slack, Teams, or email—so the right people hear about it fast.

### Deep Issue Tracker Integration

Push crashes directly into Jira, GitHub, Azure DevOps, and more. Jira users get full bi-directional sync—close the issue there, it closes in BugSplat. All issues include call stacks, metadata, and links back to the original crash report.

### Crash Overview Reports

BugSplat delivers a daily or weekly summary of your project's stability, grouped by crash type and sorted by frequency. Great for standups, status updates, or understanding how things are trending over time.

### Full Crash History

Access all your crash data for up to a year. Compare issues across builds, review long-tail defects, and make sure older bugs stay fixed.

### Easy Symbol Servers

Connect to your external symbol servers to resolve call stacks in complex or multi-platform environments. Great for large teams with sophisticated pipelines.

### Serious Security and Data Protection

SOC 2 Type 2 and GDPR compliant. Control what gets collected and who sees it—without jumping through hoops. Built to make your compliance team smile.
