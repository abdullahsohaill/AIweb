## Frequently Asked Questions

### How to view local logs?

Click Help ---> Open Logs Folder to open the local folder, find today's log zip package and send it.

### Solution for Remote SSH "Could not establish connection to" failure

According to the VSCode official documentation at VSCode FAQ , starting from version 1.99, the remote development feature mandatorily requires the remote host to have glibc>=2.28.
[LINK: VSCode FAQ](https://code.visualstudio.com/docs/remote/faq#_can-i-run-vs-code-server-on-older-linux-distributions)

### How to select and import Figma images into the IDE?

Developers need to apply for Figma permissions first. Click on the corresponding image, which requires logging in with a Tencent Enterprise WeChat email. During usage, follow the image operations below and add them to the conversation.

### How to create a new conversation or view conversation history?

Click the + button on the left to create a new conversation, and the right side shows the conversation history.

### What should I do if I see a "version incompatible" message when searching for and installing plugins in the marketplace?

Search for "gallery" in the IDE settings, then configure https://marketplace.visualstudio.com and restart the IDE.

### Terminal cannot execute related commands

The error is as follows, indicating that the shell integration switch needs to be set to Enable. Search for "shell" in settings, enable Terminal Integrated Shell Integration to Enable.

### Unable to install C/C++ extension

Solution: Install a specific lower version, or install clangd.

### How to configure the theme?

Click on Theme, select Configure Theme. Available theme options include Color Theme, File Icon Theme, and Product Icon Theme.

### How can I quickly open the IDE development window via command shortcuts in the terminal?

Use Up Arrow ↑ + Command + P to open the installation window, type "buddy" to install buddy. After successful installation, execute related commands in the terminal.
